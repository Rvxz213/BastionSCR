Privacy Policy

Last updated: July 7, 2026

This Privacy Policy explains what data Bastion ("the Bot") collects and processes when used in your Discord server.

1. Data We Store

The Bot stores minimal configuration data required for its features to work:

  • Server ID (guild ID)
  • Log channel ID you set via /setup
  • Trap channel ID you set via /trap

This data is stored in a local configuration file (settings.json) on the server where the Bot runs.

2. Data Processed Temporarily

  • Message content & attachments — processed in real time to detect spam, phishing, and trap channel violations. For logging purposes, a message summary (including text content) may be sent to the log channel you designate within your own server.
  • Message timestamps & user IDs — held temporarily in memory (not a permanent database) to detect flooding. This data is automatically cleared when the Bot restarts.

3. Data We Do NOT Collect

  • We do not sell or share your data with third parties.
  • We do not store a full history of user messages in an external database.
  • We do not collect personal data beyond public Discord information (username, ID, tag).

4. Purpose of Processing

All data is processed solely to run moderation and logging functions: detecting spam/phishing, managing the trap channel, and logging activity to a log channel that you control.

5. Storage & Retention

Configuration data is stored as long as the Bot is in your server. Logs sent to your log channel are fully under your control and subject to your own server settings.

6. Data Deletion

You can delete all configuration data related to your server by removing (kicking) the Bot from your server. For manual deletion requests, contact f0reest on Discord.

7. Third-Party Services

The Bot runs on the Discord platform and is subject to the Discord Privacy Policy.

8. Changes to This Policy

This Privacy Policy may be updated at any time. Changes take effect once published on this page.

9. Contact

For questions or data-related requests, contact f0reest on Discord.

Privacy Policy

Terakhir diperbarui: 7 Juli 2026

Kebijakan Privasi ini menjelaskan data apa yang dikumpulkan dan diproses oleh Bastion ("Bot") saat digunakan di server Discord Anda.

1. Data yang Kami Simpan

Bot menyimpan data konfigurasi minimal yang diperlukan agar fitur berfungsi:

  • ID Server (guild ID)
  • ID channel log yang Anda tetapkan lewat /setup
  • ID trap channel yang Anda tetapkan lewat /trap

Data ini disimpan dalam file konfigurasi lokal (settings.json) di server tempat Bot dijalankan.

2. Data yang Diproses Sementara

  • Isi pesan & lampiran — diproses secara real-time untuk mendeteksi spam, phishing, dan pelanggaran trap channel. Untuk keperluan logging, ringkasan pesan (termasuk isi teks) dapat dikirim ke channel log yang Anda tentukan di dalam server Anda sendiri.
  • Timestamp pesan & ID pengguna — disimpan sementara di memori (bukan database permanen) untuk mendeteksi flooding. Data ini otomatis hilang saat Bot dimulai ulang.

3. Data yang TIDAK Kami Kumpulkan

  • Kami tidak menjual atau membagikan data Anda ke pihak ketiga.
  • Kami tidak menyimpan riwayat lengkap pesan pengguna dalam database eksternal.
  • Kami tidak mengumpulkan data pribadi di luar informasi publik Discord (username, ID, tag).

4. Tujuan Pemrosesan

Semua data diproses semata-mata untuk menjalankan fungsi moderasi dan logging: mendeteksi spam/phishing, mengelola trap channel, dan mencatat aktivitas ke channel log yang Anda kontrol.

5. Penyimpanan & Retensi

Data konfigurasi disimpan selama Bot berada di server Anda. Log yang dikirim ke channel log Anda sepenuhnya berada di bawah kendali Anda dan tunduk pada pengaturan server Anda sendiri.

6. Penghapusan Data

Anda dapat menghapus semua data konfigurasi yang berkaitan dengan server Anda dengan mengeluarkan (kick) Bot dari server. Untuk permintaan penghapusan manual, hubungi f0reest di Discord.

7. Layanan Pihak Ketiga

Bot berjalan di atas platform Discord dan tunduk pada Kebijakan Privasi Discord.

8. Perubahan Kebijakan

Kebijakan Privasi ini dapat diperbarui sewaktu-waktu. Perubahan berlaku sejak dipublikasikan di halaman ini.

9. Kontak

Untuk pertanyaan atau permintaan terkait data, hubungi f0reest di Discord.
