## ✨ Features

### 🛡 Security
- Anti Spam
- Anti Phishing
- Scam Link Detection
- Raid Protection
- Auto Moderation

### 📜 Logging
- Member Join Logs
- Member Leave Logs
- Audit Logs
- Message Delete Logs
- Message Edit Logs

### 👋 Community
- Welcome Messages
- Goodbye Messages
- Auto Role Assignment

### ⚡ Performance
- Discord.js v14
- Modular Architecture
- Lightweight & Fast
- Easy Configuration

# 🚀 Installation

```bash
git clone https://github.com/USERNAME/BastionSecurityBot.git
```

```bash
cd BastionSecurityBot
```

```bash
npm install
```

```bash
npm start
```

---

# ⚙️ Requirements

- Node.js 18+
- Discord Bot Token
- Discord Developer Portal Application

---

# 🔧 Configuration

Create a `.env` file.

```env
TOKEN=YOUR_DISCORD_TOKEN
CLIENT_ID=YOUR_CLIENT_ID
GUILD_ID=YOUR_GUILD_ID
```
